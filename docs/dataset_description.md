# Dataset Description

## Overview
This dataset contains information related to greenhouse gas (GHG) emissions,
including geographic coordinates, population data, and total emissions measured
in COâ‚‚ equivalent units.

The dataset is used for analytical and visualization purposes within a
Databricks SQL dashboard.

## Data Source
Public dataset used for learning and portfolio demonstration purposes.

## Level of Aggregation
The data is aggregated at different geographic levels, including:
- Country
- State
- County

## Key Fields Used
| Column Name | Description |
|------------|-------------|
| latitude | Geographic latitude |
| longitude | Geographic longitude |
| emissions | Total greenhouse gas emissions |
| population | Population count |
| state_abbr | State abbreviation |
| county_name | County name |

## Notes and Limitations
- The dataset is used for educational purposes only.
- Emissions values may represent aggregated estimates.
- No sensitive or private data is included.
